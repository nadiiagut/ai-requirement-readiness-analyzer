## Usage in n8n

1. Trigger: Jira Issue Created (or scheduled)
2. HTTP Request: POST to `/analyze/confluence-page?demo_mode=false`
3. Extract: `$.page_title` and `$.page_body`
4. Confluence: Create Page with extracted title and body
5. Optional: Link page back to Jira issue
